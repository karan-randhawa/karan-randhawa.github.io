Collaborations Management Application

Collaber is a UWP App that helps users manage Projects, their Tasks and Collaborators in school or at work. It uses Facebook Authentication, and can help users maintain a to-do list for their projects. Some of it's core-features are:
1) Checkable Tasks - Users as well as their collaborators can check/uncheck Tasks and others will see an updated version in real-time.
2) Add Collaborators using Facebook Email - Collaborators can be added using their Facebook-registered Email address, they will be presented with the Projects they have been added to, upon next login or initial signup.
3) Cross-device - Use it on any device that runs on Windows 10. Users can find all their data synced across all the devices they are logged into, using the same Facebook account.

Technologies:
C#, XAML, Windows 10 SDK, Microsoft Azure

Marvel Fan App - Character Explorer

Hero Edge is a very simple, yet effective UWP App focused on the Client-Side. Upon launch, it presents the user with a random number of characters from Marvel, collected from the Marvel API. Users can then view details of each Character and the associated Comics they have. Users can also click on a particular Comic to see it's details and a short description and price, if available.

Technologies:
C#, XAML, Windows 10 SDK
